This directory is for game assets.

Suggested assets to add:
- dragon.png - Purple dragon sprite
- meteorite.png - Meteorite obstacle sprite
- jewel.png - Collectible jewel sprites (different colors)
- background.png - Sky background image
- sounds/collect.wav - Sound effect for collecting jewels
- sounds/explosion.wav - Sound effect for hitting meteorites
- sounds/music.mp3 - Background music

You can replace the simple shape drawings in the game code with these image assets when available.
